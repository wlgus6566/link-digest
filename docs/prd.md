# 📄 **PRD: LinkDigest**

## ✅ 1. 제품 개요

**제품 이름 (임시)**: LinkDigest

**제품 목적**:

링크 기반 콘텐츠(YouTube, Instagram 등)의 정보를 AI가 요약 및 시각적으로 정리해주고, 이를 저장하여 나중에 쉽게 찾아볼 수 있도록 지원하는 앱입니다.

**해결하고자 하는 문제**:

- 영상/이미지 중심 콘텐츠의 핵심 내용을 빠르게 파악하기 어려움
- 콘텐츠를 북마크해도 정리되지 않아 나중에 다시 보기 불편함
- 메모, 캡처 등 수작업 정리가 번거로움

**제공하는 가치**:

- AI 요약과 이미지 추출로 콘텐츠를 ‘블로그 포스트’처럼 시각화
- 정리된 콘텐츠를 태그/카테고리로 저장, 검색 가능
- 언제든 다시 볼 수 있는 나만의 콘텐츠 아카이브 제공

## 👥 2. 대상 사용자

| 주요 사용자군                         | 니즈 및 문제점                                               |
| ------------------------------------- | ------------------------------------------------------------ |
| 자기계발에 관심 있는 직장인           | 유튜브 강연, 인사이트 콘텐츠를 많이 소비하지만 정리가 어려움 |

| 인플루언서 및 블로거                  | 콘텐츠 소스를 체계적으로 정리하고, 재가공하여 활용하고자 함  |

## 🔄 3. 사용자 흐름

### 📽️ 시나리오: 유튜브 영상 요약 저장하기

1. 사용자가 앱에서 ‘링크 추가’ 버튼 클릭
2. 유튜브 URL 입력 → 콘텐츠 분석 자동 시작
3. AI가 썸네일 및 주요 장면 이미지 추출
4. 영상 자막 기반으로 요약문 생성
5. 태그/카테고리 자동 추천 → 사용자 수정 가능
6. 콘텐츠 저장 후 내 보관함에서 열람 가능

### 📁 시나리오: 저장한 콘텐츠 다시 보기

1. 사용자가 내 보관함 접근
2. 검색창 또는 필터(플랫폼/태그 등) 사용
3. 콘텐츠 클릭 → 정리된 블로그 형식으로 열람

## 🧪 5. 기술 스택

### 🖥️ Frontend

| 항목        | 내용                                            |
| ----------- | ----------------------------------------------- |
| 프레임워크  | Next.js (App Router, TypeScript)                |
| 스타일링    | Tailwind CSS + Shadcn UI                        |
| 상태관리    | React Context API (MVP 범위 내에서는 경량 구조) |
| 이미지 처리 | Next.js `<Image>` 컴포넌트 사용                 |
| 배포        | Vercel                                          |
| 인증 연동   | Supabase Auth 사용 (OAuth 또는 Email/Password)  |

---

### ⚙️ Backend (Supabase 기반)

| 항목          | 내용                                                |
| ------------- | --------------------------------------------------- |
| 데이터베이스  | Supabase PostgreSQL                                 |
| 인증          | Supabase Auth (JWT 기반)                            |
| 파일 저장     | Supabase Storage (썸네일, 대표 이미지 저장)         |
| 서버리스 함수 | Supabase Edge Functions (AI 요약 호출 시 래핑 가능) |
| 보안 정책     | RLS (Row-Level Security)로 유저별 접근 제한         |
